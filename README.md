# 🧪 React Testing Playground (Vite + React + TypeScript)

Repositorio enfocado en pruebas de integración utilizando herramientas modernas como **Vitest, React Testing Library y Jest**.

Este proyecto demuestra buenas prácticas actuales para testing en aplicaciones React, utilizando **Vite** como bundler para lograr un entorno rápido, moderno y eficiente.

---

## 🚀 Tecnologías utilizadas

- ⚛️ React
- ⚡ Vite
- 📘 TypeScript
- 🧪 Vitest
- 🧫 React Testing Library
- 🔍 Jest (compatibilidad y conceptos)
- 🎯 ESLint + Prettier

---

## 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/rineliniguezsosa/first-steps-with-testing.git
```

Accede al proyecto:

```bash
cd first-steps-with-testing
```

Instala las dependencias:

```bash
npm install
```

## Ejecutar el proyecto

```bash
npm run dev
```

Luego abre en tu navegador:

```bash
http://localhost:5173
```

## Ejecutar pruebas

```bash
npm run test
```
