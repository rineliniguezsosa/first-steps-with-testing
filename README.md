# 🧪 React Testing Playground (Vite + React + TypeScript)

Repositorio enfocado en pruebas de integración utilizando herramientas modernas como **Vitest, React Testing Library y Jest**.

Este proyecto demuestra buenas prácticas actuales para testing en aplicaciones React, utilizando **Vite** como bundler para lograr un entorno rápido, moderno y eficiente.

---

## 🚀 Tecnologías utilizadas

- ⚛️ React
- ⚡ Vite
- 📘 TypeScript
- 🧪 Vitest
- 🧫 React Testing Library
- 🔍 Jest (compatibilidad y conceptos)
- 🎯 ESLint + Prettier
